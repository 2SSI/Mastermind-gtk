# Mastermind-gtk

Graphic user interface of mastermind game in C using [GTK 3.0](https://www.gtk.org/docs/installations/index).

## Try it out

For Linux, download the third version of [GTK](https://www.gtk.org/docs/installations/index)

        make all
        ./main.exe
